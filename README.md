This project is a Municipal Public Park Facility Booking & Maintenance Request System developed using Java and JDBC.
It is a console-based application that manages public park facilities and bookings.
The system allows registering facilities and citizens, and creating facility bookings.
All data is stored and retrieved from a database using JDBC connectivity.
The application follows a layered architecture with Bean, DAO, Service, and Utility classes.
This project is suitable for learning JDBC and Java-based database applications.
